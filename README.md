## What is Cloud?

Before cloud computing, we had to rely on the strength and capacity of our own personal devices to handle files, software, and more. Now with cloud computing, we can access those files and applications from anywhere in the world without sacrificing storage or performance.

### What is AWS?

AWS (Amazon Web Services) is the leading cloud platform, which means it provides computing resources, like storage, databases, and networking, over the internet.

AWS professionals help build and maintain cloud applications. Some popular cloud career paths include Cloud Architect, DevOps, Security, and  Data Engineers. Today, I'm most interested in Cloud Engineering since they tend to wear a lot of hats!
